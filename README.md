Attendance App
Overview
The Attendance Tracker App, built with Flutter, revolutionizes attendance management by eliminating the need for manual updates, Excel sheets, and paperwork. This app empowers management to effortlessly track daily student attendance, swiftly update changes in bus routes, schools, and student information within seconds. It also provides the flexibility to create, modify, or delete records as required, streamlining administrative tasks and enhancing efficiency.

Features
Automated Attendance Tracking:

Eliminate manual updates and paperwork by automating the tracking of daily student attendance.
Effortless Updates:

Quickly update changes within bus routes, schools, and student information within seconds.
Dynamic Management:

Create, modify, or delete records as needed, providing dynamic control over attendance and information management.
